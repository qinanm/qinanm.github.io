delete删除的是什么

发表于2022-08-22|更新于2022-08-22|js
|阅读量:

深入理解 JavaScript 中的 delete 操作符
delete - 掘金

delete操作符用于删除对象的属性。某些属性不能被删除的根本原因在于：这些属性拥有内部属性DontDelete,该内部属性控制着该属性是否可以被删除

  • 通过关键字声明的变量带有DontDelete属性, 所以无法删除

  • 未声明的变量不带有内置属性（ReadOnly、DontEnum、DontDelete 和 Internal），所以可以直接删除

  • 删除的是对象的属性，而不是对象的值

  • ，eval 在处理变量声明时有一个特殊的行为：在 eval 中声明的变量事实上没有 DontDelete 属性

  • 在 eval 代码中的函数内部通过 var 定义的变量具有 DontDelete，不能被删除。

  • 内置对象的一些属性拥有内部属性DontDelete,因此不能被删除

  • 特殊的arguments变量（活化对象的属性）拥有DontDelete,(函数参数也有，不能删)因此不能被删除

  • 任何函数实例的length(返回形参长度)属性也拥有DontDelete

  • 原型 prototype 中声明的属性和对象自带的属性(其实这些属性也是在原型 prototype 中的)可以认为是带有 DontDelete 特性的，无法被删除。
