CSS选择符有哪些？哪些属性可以继承？

发表于2022-08-05|更新于2022-08-05|css
|阅读量:

常用选择器

  • ID选择器
  • 类选择器
  • 标签选择器
  • 相邻选择器（h1 + p）
  • 子选择器（ul>li）
  • 后代选择器（li a）
  • 通配符选择器
  • 属性选择器（a[rel = "external"]）
  • 伪类选择器（a:hover, li:nth-child(even)）

继承问题

文字相关的属性一般可以继承, 如font-size font-family color, UL LI DL DD DT;
盒模型相关的属性一般不能继承，如border padding margin width height
