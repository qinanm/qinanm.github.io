okk，看了一下官网关于chrome插件开发，通过在manifest.json中配置content_scripts,并且如果需要将代码注入页面，扩展必须有cross-origin权限，还必须可以使用chrome.tabs模块，这些可以通过在manifest.json配置permissions字段来声明取得这些权限

一旦设置好了权限就可以通过调用executeScript()来注入js脚本，通过insertCSS()来注入css

遇到报错：Uncaught (in promise) DOMException: play() failed because the user didn’t interact with the document

DOMException: play()失败，因为用户没有首先与文档交互。

报错原因：

浏览器为了提高用户体验，减少性能消耗，只有在发生用户行为的时候才可以播放音频或视频。而页面加载，定时请求不属于用户行为，因此被禁止自动播放
