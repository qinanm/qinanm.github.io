移除元素

  1. 题目概述

    • 参数：一个数组nums和一个值val
    • 条件：原地移除nums内值为val的元素，不使用额外的数据空间（元素顺序可以改变，不需要考虑超出新长度后面的元素）
    • 返回：新数组的长度

  2. 思路

    • 设置一个指针i, 永远指向前面等于val的元素
    • 遇到 === val 指针指向index
      • index = 0 –> i=index
      • index > 0 —> nums[index-1] !== val 时 i指向当前index,相等时指针不变
    • 遇到 !== val 指针 与 指针 当前指向 交换元素， 指针后移
    • i永远指向 === val的元素 或 !==val 的下一个未知的元素

  3. 解决 方案

    1. 方法： 使用forEach ，总觉得搞麻烦了（对这题来说）

    2. 方法：使用for循环，试试会不会优化一点，思路不变

      • 速度快来一些，空间还是很拉，我觉得变快的主要原因是 使用forEach 时 === val用了两个判断条件来使i=index, 而for循环则使用一个条件利用continue是 i != index 直接跳过，即判断两个满足条件和排除一个不满足条件的区别
