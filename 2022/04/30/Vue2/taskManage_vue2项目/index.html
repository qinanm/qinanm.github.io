一、项目前期配置准备

1.创建一个项目

用到 vue-cli 脚手架

  1. vue2 项目选择 vue2

  1. 选择 use npm， 然后等待配置完成

任务管理

3.项目技术选型

  • 使用 vue-cli 创建项目， 使用 vue2 写项目
  • 使用 vue-router 配合创建单页面应用
  • 使用 vuex 集中管理数据资源
  • 使用 axios 发送请求获取数据
  • 组件的数据的交互需要好好复习
  • 使用 element-ui 来加快页面的开发
  • 使用 localstorage 进行数据的存储
  • 使用 css 预处理器 less 来写样式
  • 使用 nanoid 作为唯一标识

4.项目具体功能

任务管理页面

  1. 关于任务，可以添加任务，修改任务，删除任务，删除已完成任务，全选完成任务
  2. 实现一个日历（半永久）
  3. 实现一个时钟展示 （半永久）

心情监测页面

  1. 心情对应的 emoji 轮播图展现
  2. 心情报告 — 统计不同心情次数 体现一周主情绪

来点 music（需要一个音乐接口，如果没有的话，考虑一下爬虫获取）

  2. 搜索框防抖功能
  3. 随机获取音乐
  4. 播放音乐功能
  5. （先放着，可能做不来）， 跟着音乐律动的线条

  2. 加个美图吧
  3. 每日固定时间更新美句+美图

  5. 给美图加个放大镜效果（添加理由，学了这个东西，就顺手做出来试试）
